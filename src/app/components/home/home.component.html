Cel

Zaprojektowanie i implementacja w Angular aplikacji front-end komunikującej się z API http://www.recipepuppy.com/about/api/ Należy utworzyć stronę, która będzie miała w formie listy 20 przykładowych składników kulinarnych (nazwy angielskie). Po wybraniu jednej z nich użytkownik powinien mieć przycisk Sprawdź przepisy. Jego wybranie powinno skutkować odpytaniem API i zwróceniem listy przepisów dostępnych z wykorzystaniem wybranego składnika lub wybranych składników (grupowo). Lista powinna zaprezentować miniaturkę przepisu a także tytuł dania do stworzenia.

Wymagania

Interfejs użytkownika

Aplikacja Angular/HTML/CSS umożliwiająca podgląd pobranych danych z webService. Aplikacja powinna być responsywna w oparciu o Bootstrap lub flexbox.
Aplikacja powinna w sposób czytelny i estetyczny prezentować informację o przepisach pobranych z API.
Aplikacja powinna odpowiednio zawijać się w trybie mobilnym.

Zadanie 2

Z lokalnego serwera API wczytać przykładowy dokument JSON, w którym będzie można w jakimś określonym formacie wpisać nazwę kolumny (nagłówek) oraz typ danych i jej wartość. Następnie aplikacja powinna wczytać zawartość tego komunikatu API i odrysować na stronie tabelkę odpowiadającą strukturze z tego pliku z formatowaniem danych. Jako formaty przyjmujemy dla uproszczenia typ liczba i tekst.

Język programowania

Angular, HTML 5, Bootstrap
